# Oftps
An Online File Transfer Protocol client Service.

### Owner/Creator
CDNievas - Web designer/developer.

### Objetive
Create an online client for FTP servers.

### Goals
- [x] Connect to FTP server.
- [ ] View of filesystem.
- [ ] Create folders.
- [ ] Rename files.
- [ ] Delete files.
- [ ] Upload multiple files.
- [ ] Change permissions of files.
- [ ] Move files.
- [ ] Drag and drop system to upload multiple files.
- [ ] Online console to execute commands.
> - [ ] Full compatibility with cell phones (Responsive Design).
> State of compatibility (Responsive Design): 100%
- [ ] Modify txt files online (?)
